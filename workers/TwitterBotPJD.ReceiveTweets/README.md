# TwitterBotPJD.ReceiveTweets

This project requires the user secrets to be set up as follows: 
```json
{
    "TwitterCredentials": {
      "AccessToken": "***************************",
      "AccessTokenSecret": "******************************",
      "ConsumerKey": "**********************",
      "ConsumerSecret": "*************************",
      "BearerToken": "**********************************************************************"
    }
}
```
